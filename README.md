# AutoMirei

Cxkitty学习通自动刷课的安卓封装，修改自**Python IDE KtxPy**。

## 特性

- [x] Python 3.11
- [x] 扫码登录，密码登录
- [x] 获取课程列表，自动刷课
- [ ] 支持多会话刷课
- [ ] 支持多账号刷课
- [ ] 封装对接刷题题库
- [ ] 支持自定义刷课时间
- [ ] 支持多会话

## 使用

点击右下角创建会话（目前不支持多会话），呼出键盘操作CxKitty

# KtxPy

Developed To Run Python On Android With Material3 Support
